Micro Interactions for Angular: ng-micro-interact 

    Directive: ngMicroInteract
  
    Inputs:

    type: (default = 'fade') string - The type of micro interaction (blink, groove, jerk etc..)
    trigger: (default = 'mouseover') string - Event name
    ** options: (optional, default duration = 2000) KeyframeAnimationOptions could include etc: duration
